Project A

Project A stands for project angelascension. It is a video game with the basic premise being an og Zelda  undertale inspired game where you play as the test subject “Taz” who was raised in a lab to be the vessel for an angel called Tazriel. Being the angel’s vessel the play has access to powerful angelic abilities which when used fill a “progress bar” (ascension), throughout the game using these special abilities slowly fills your ascension until the playable character transforms into an angel at 100% ascension. The game will then present a plot twist as becoming an angel will be considered the bad ending.
